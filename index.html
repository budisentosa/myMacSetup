Complete Installation Guide

1. Brew & Basic CLI

# Install Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Core Tools
brew install git fish starship oh-my-posh neovim

Terminal & Core

brew install --cask ghostty
brew install --cask visual-studio-code
brew install git

System & Productivity

brew install --cask displaylink
brew install --cask raycast
brew install --cask karabiner-elements

Window Management

# Yabai & Skhd
brew install koekeishiya/formulae/yabai
brew install koekeishiya/formulae/skhd

Infrastructure & AI

# Container & AI CLI
brew install orbstack
npm install -g @anthropic-ai/claude-code
npm install -g @google/gemini-cli

2. SSH & Private Dotfiles

Cloning your private dotfiles and setting up system links:

# 1. Clone dotfiles
gh auth login
git clone https://github.com/budisentosa/dotfiles3.git ~/Documents/dotfiles3

# 2. Set up System Links (Symlinks)
ln -s ~/Documents/dotfiles3/.ssh/config ~/.ssh/config
ln -s ~/Documents/dotfiles3/fish/config.fish ~/.config/fish/config.fish
ln -s ~/Documents/dotfiles3/nvim/init.lua ~/.config/nvim/init.lua
